# PRD - Quant Data Platform

## Overview

Quant Data Platform is a data engineering portfolio project demonstrating a reproducible market data pipeline.

## Goals

- Download historical market data from Binance (BTC/ETH, 1h / 4h / 1d intervals)
- Validate and clean raw datasets
- Compute technical indicators (SMA, EMA, RSI)
- Generate analytics reports and charts

## Success Criteria

1. Download BTCUSDT historical data from Binance
2. Store raw data locally
3. Clean and validate the dataset
4. Generate SMA, EMA, and RSI indicators
5. Produce charts and summary statistics
6. Run the entire workflow locally from the command line
